\section{Recommendation for Future Work}

\todo{建议篇幅：350--500 words，5 段或 5 点。Future work 的优先级要服务新主线：先补强 baselines 和 closed-loop robustness，再扩展到 real robot / heterogeneous setting。
\begin{enumerate}
  \item P1: Add stronger output-interface baselines。补充 direct action head、WAM plus IDM/readout、same-backbone IDM 等对照，以验证优势是否来自 action-in-pixel interface。
  \item P2: Improve closed-loop robustness。系统研究 CFG selection、on-policy generated data、sim-context rollout、consistency filtering and action decoder finetuning。
  \item P3: Conduct real robot evaluation。在真实机器人平台上验证 closed-loop task success。
  \item P4: Scale the video-generation policy。尝试更大 WAN/video backbone、更多 robot demonstration data 和更系统的 checkpoint sweep，检验 video pretraining 能力是否继续转化为 policy improvement。
  \item P5: Future extension to heterogeneous/action-free data。把更多机器人平台或人类视频放入同一 visual action representation，但明确这是后续工作，不是本 thesis 的主要实验证明。
\end{enumerate}}
